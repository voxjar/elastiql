pub struct Cursor(_);

An opaque cursor (obtained from the cursor field on an Edge) to return results after. This facilitates results pagination according to the Relay Cursor Connections Specification.

Note: it is expected that this will be used in conjunction with the same sort argument that created this Cursor.
